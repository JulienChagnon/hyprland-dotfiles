# 󰌌  Hyprland Keybindings

## Apps & Launchers

| Keys                  | Action                       |
|-----------------------|------------------------------|
| `SUPER` + `Q`         | Terminal (kitty)             |
| `SUPER` + `E`         | File manager                 |
| `SUPER` + `R`         | App launcher (menu)          |
| `SUPER` + `SPACE`     | Launcher                     |
| `SUPER` + `A`         | App launcher (icon grid)     |
| `SUPER` + `C`         | Claude AI                    |
| `SUPER` + `W`         | Wallpaper picker             |
| `SUPER` + `O`         | Waybar theme picker          |
| `SUPER` + `L`         | Lock screen                  |
| `SUPER` + `M`         | Power menu (shutdown/restart/sleep) |
| `SUPER` + `D`         | Rnote (sketch canvas)        |
| `Print`               | Screenshot region → annotate |

## Windows

| Keys                            | Action                       |
|---------------------------------|------------------------------|
| `SUPER` + `X`                   | Close window                 |
| `SUPER` + `F`                   | Toggle floating              |
| `SUPER` + `Z`                   | Fullscreen (hides waybar)    |
| `SUPER` + `P`                   | Pseudo-tile (dwindle)        |
| `SUPER` + `J`                   | Toggle split direction       |
| `SUPER` + `←/↓/↑/→`            | Move focus                   |
| `SUPER` + `SHIFT` + `←/↓/↑/→`  | Move window                  |

## Workspaces

| Keys                       | Action                       |
|----------------------------|------------------------------|
| `SUPER` + `,` / `.`        | Cycle workspaces             |
| `SUPER` + `S`              | Toggle scratchpad            |
| `SUPER` + `SHIFT` + `S`    | Send to scratchpad           |

## System

| Keys                   | Action                       |
|------------------------|------------------------------|
| `SUPER` + `H`          | Show this cheatsheet         |
| `SUPER` + `SHIFT` + `R`| Reload Hyprland config       |
| `XF86AudioRaise/Lower` | Volume up / down             |
| `XF86AudioMute`        | Mute toggle                  |
| `XF86MonBrightness±`   | Brightness                   |
